
# 📘 Assignment: Hangman Game

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Develop a fully functional Hangman game that randomly selects a word from a predefined list and allows players to guess letters until they either solve the word or run out of attempts.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display the current progress in underscore format (_ _ _ )
- Track incorrect guesses remaining and display them to the player
- Check if each guessed letter is in the word and update the progress accordingly
- End the game when the word is guessed or all attempts are exhausted
- Display win/lose messages with the final word revealed
